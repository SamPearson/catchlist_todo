====== Metric Type Routes ======

===== Overview =====

API endpoints for managing metric types. Metric types define what measurements a user wants to track on their reports.

Examples: "Sleep Hours", "Work RPE", "Diet Adherence", "Exercise Completed"

All endpoints require authentication via JWT Bearer token.


===== Endpoints =====

==== List Metric Types ====

**GET /api/metric-types**

Returns all metric types for the authenticated user.

| Parameter | Type | Location | Required | Description |
|-----------|------|----------|----------|-------------|
| include_archived | boolean | query | no | If true, includes archived metric types. Default false. |

**Response:** Array of metric type objects, ordered by sort_order then name


==== Create Metric Type ====

**POST /api/metric-types**

Creates a new metric type.

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| name | string | yes | Display name (max 100 chars) |
| value_type | string | yes | Data type: integer, float, rating, boolean |
| unit | string | no | Label for display (e.g., "hours", "1-10") |
| min_value | integer | no | Minimum valid value (for rating type) |
| max_value | integer | no | Maximum valid value (for rating type) |
| sort_order | integer | no | Default display ordering |

**Response:** Created metric type object (201)

**Example Request (simple):**
	name: "Sleep Hours"
	value_type: "integer"
	unit: "hours"

**Example Request (rating):**
	name: "Work RPE"
	value_type: "rating"
	unit: "1-10"
	min_value: 1
	max_value: 10


==== Get Metric Type ====

**GET /api/metric-types/<metric_type_id>**

Returns a specific metric type by ID.

**Response:** Metric type object, or 404 if not found


==== Update Metric Type ====

**PUT /api/metric-types/<metric_type_id>**

Updates a metric type.

| Field | Type | Description |
|-------|------|-------------|
| name | string | Display name (cannot be empty) |
| value_type | string | Data type: integer, float, rating, boolean |
| unit | string | Label for display |
| min_value | integer | Minimum valid value |
| max_value | integer | Maximum valid value |
| sort_order | integer | Display ordering |

**Response:** Updated metric type object, or 404 if not found


==== Delete Metric Type ====

**DELETE /api/metric-types/<metric_type_id>**

Permanently deletes a metric type. **Warning:** This cascades to all MetricValues using this type. Prefer archiving to preserve historical data.

**Response:** 204 No Content, or 404 if not found


==== Archive Metric Type ====

**POST /api/metric-types/<metric_type_id>/archive**

Archives a metric type (soft delete). Archived metrics are hidden from templates but historical data is preserved.

**Response:** Updated metric type object with active=false, or 404 if not found


==== Reactivate Metric Type ====

**POST /api/metric-types/<metric_type_id>/reactivate**

Reactivates an archived metric type.

**Response:** Updated metric type object with active=true, or 404 if not found


===== Metric Type Object =====

| Field | Type | Description |
|-------|------|-------------|
| id | integer | Primary key |
| user_id | integer | Owner |
| name | string | Display name |
| value_type | string | Data type: integer, float, rating, boolean |
| unit | string | Label for display |
| min_value | integer | Minimum valid value |
| max_value | integer | Maximum valid value |
| active | boolean | False if archived |
| sort_order | integer | Display ordering |
| created_at | datetime | When created |
| updated_at | datetime | When last modified |


===== Value Types =====

| Type | Description | Example |
|------|-------------|---------|
| integer | Whole number | Sleep hours: 7 |
| float | Decimal number | Weight: 185.5 |
| rating | Bounded integer (uses min/max) | Work RPE: 8 (1-10 scale) |
| boolean | Yes/no | Exercise completed: true |
