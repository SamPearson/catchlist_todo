====== Checkins ======
Created Friday 04 April 2026


===== Priority 1: Smoke Tests (Critical Path) =====

These validate that basic checkin functionality works at all.

==== ✅️Smoke Test List ====
1. **Create a basic checkin** - POST /api/checkins with target_type, target_id, note
2. **List checkins for a target** - GET /api/checkins/target?target_type=task&target_id=123 (verify checkin appears)
3. **Get checkin by ID** - GET /api/checkins/{id}
4. **Update checkin note** - PUT /api/checkins/{id}
5. **Delete checkin** - DELETE /api/checkins/{id}

===== Priority 2: Business Logic Tests =====

==== Occurred_at vs Created_at Behavior ====
76. **Occurred_at represents when event happened** - Create checkin with occurred_at yesterday, verify occurred_at is yesterday
77. **Created_at represents when recorded** - Create checkin with occurred_at yesterday, verify created_at is now
78. **Occurred_at can be backdated** - Verify occurred_at < created_at is valid
79. **Occurred_at defaults to now** - Omit occurred_at, verify occurred_at ≈ created_at

==== Timezone Handling ====
80. **List checkins converts UTC to local time** - Verify occurred_at in response is local
81. **Get checkin converts UTC to local time** - Verify all timestamps in response are local
82. **Create checkin accepts local time** - Verify occurred_at input treated as local
83. **Update checkin accepts local time** - Verify occurred_at input treated as local
84. **Timezone uses user default when not configured** - Verify fallback to UTC

==== Ordering Logic ====
85. **Checkins ordered by occurred_at DESC primary** - Create 5 checkins with different occurred_at, verify order
86. **Checkins ordered by created_at DESC secondary** - Create 3 checkins with same occurred_at, verify created_at order
87. **Ordering handles null occurred_at (if allowed)** - Verify nulls sorted appropriately (if applicable)

==== Target Type Variations ====
88. **Multiple checkins can exist for same target** - Create 5 checkins for same task, verify all listed
89. **Checkins for different targets are isolated** - Create checkins for task and project, verify filtering works
90. **Checkins support all target types** - Create checkin for each target type, verify all work
