====== Tags ======

===== Priority 1: Smoke Tests (Critical Path) =====

These validate that basic tag functionality works at all.

==== Smoke Test List ====
1. **Create a basic tag** - POST /api/tags with only name
2. **List tags** - GET /api/tags (verify tag appears)
3. **Get tag by ID** - GET /api/tags/{id}
4. **Update tag name** - PUT /api/tags/{id}
5. **Delete tag** - DELETE /api/tags/{id}


===== Priority 2: Extended CRUD Tests =====

==== List Tags (GET /api/tags) ====
6. **List tags returns empty array** - New user with no tags gets []
7. **List tags only returns user's own tags** - Create tags for two different users, verify isolation
8. **List tags returns all tags** - Create 5 tags, verify all returned (no filtering)

==== Get Tag (GET /api/tags/{id}) ====
