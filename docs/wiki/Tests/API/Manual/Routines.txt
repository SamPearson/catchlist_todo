====== Routines ======
Created Tuesday 31 March 2026


===== Priority 1: Smoke Tests (Critical Path) =====

These validate that basic routine functionality works at all.

==== ✅️Smoke Test List ====
1. **Create a basic routine** - POST /api/routines with only title
2. **List routines** - GET /api/routines (verify routine appears)
3. **Get routine by ID** - GET /api/routines/{id}
4. **Update routine title** - PATCH /api/routines/{id}
5. **Delete routine** - DELETE /api/routines/{id}
