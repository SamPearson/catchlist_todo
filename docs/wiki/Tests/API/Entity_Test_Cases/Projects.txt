====== Projects ======
Created Thursday 19 March 2026


===== Priority 1: Smoke Tests (Critical Path) =====

These validate that basic project functionality works at all.

==== ✅️Smoke Test List ====
1. **Create a basic project** - POST with only title
2. **List projects** - GET /api/projects (verify project appears)
3. **Get project by ID** - GET /api/projects/{id}
4. **Update project title** - PUT /api/projects/{id}
5. **Delete project** - DELETE /api/projects/{id}


===== Priority 2: Extended CRUD Tests =====

==== ✅️List Projects (GET /api/projects) ====
6. **List projects excludes completed by default** - Create completed and incomplete projects, verify only incomplete returned
7. **List projects with include_completed=true** - Verify completed projects included when param is true
8. **List projects with include_completed=false** - Verify completed projects excluded (explicit false)
9. **List projects excludes inactive by default** - Create active and inactive projects, verify only active returned
10. **List projects with include_inactive=true** - Verify inactive projects included when param is true
11. **List projects with include_inactive=false** - Verify inactive projects excluded (explicit false)
12. **List projects with both filters combined** - Test include_completed and include_inactive together
13. **List projects returns empty array** - New user with no projects gets []
14. **List projects only returns user's own projects** - Create projects for two different users, verify isolation
