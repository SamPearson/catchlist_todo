====== Reports ======
Created Wednesday 09 April 2026


===== Priority 1: Smoke Tests (Critical Path) =====

These validate that basic report functionality works at all.

==== ✅️Smoke Test List ====
1. **Get or create report for today** - GET /api/reports/day/<date> with today's date
2. **List reports** - GET /api/reports (verify report appears)
3. **Get report by ID** - GET /api/reports/{id}
4. **Update report plan** - PUT /api/reports/{id} with plan field
5. **Delete report** - DELETE /api/reports/{id}

64. **Window scope includes soft commitments to nested timeframes** - Create monthly report with commitments to weeks/days within, verify all included
65. **Window scope includes hard commitments within timeframe** - Create hard commitments with due dates in timeframe, verify included
66. **Window scope excludes commitments outside timeframe** - Create commitments outside timeframe boundaries, verify excluded
67. **Window scope for day shows only that day** - Create daily report, verify only day's commitments
68. **Window scope for week shows all within week** - Create weekly report with commitments to multiple days, verify all included
69. **Window scope for month shows all within month** - Create monthly report with commitments to weeks and days, verify all included
70. **Window scope for season shows all within season** - Create seasonal report with commitments to months/weeks/days, verify all included
71. **Window scope for year shows all within year** - Create yearly report with commitments to all nested timeframes, verify all included

==== Commitment Scope: direct ====
72. **Direct scope shows only commitments to exact timeframe** - Create monthly report with commitments to month and to nested weeks, verify only month commitments
73. **Direct scope excludes nested timeframe commitments** - Verify commitments to child timeframes excluded
74. **Direct scope excludes hard commitments** - Create hard commitments with due dates in timeframe, verify excluded from direct scope

==== Commitment Scope: none ====

76. **None scope omits stats field entirely** - Verify stats field not in response
77. **None scope omits label field** - Verify label field not in response
